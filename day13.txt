append() , appendChild()
부모 노드의 제일 마지막 자식 요소로 추가
append()는 Node 와 String 추가 가능 , 여러가지 값을 한번에 붙일 수 있음. 반화값이 없음.
appendChild()는 Node 만 추가 가능. 추가한 Node를 반환 한다.

JQuery - 자바 스크립트의 DOM을 쉽게 작업 할 수 있도록 도와주는 프로그램
프레임 워크 - 특정 프로그램 개발의 시작부터 끝까지 필요한 모든 기능을 제공.
값 가져오기 & 설정하기 ////// $(선택자).val(“설정할 값“);
text변경하기 /////// $( 선택자 ).text( “글자” );
속성 변경하기//////$( 선택자 ).attr( “속성”, “속성값” );
html 변경하기 //////// $( 선택자 ).html( html );

$( 선택자 ).before( 요소 );
선택된 요소의 이전 형제 요소로 추가

$( 선택자 ).after( 요소 );
선택된 요소의 다음 형제 요소로 추가

$( 선택자 ).remove();
선택한 요소를 DOM 트리에서 삭제

$( 선택자 ). empty();
선택한 요소의 자식 요소를 모두 삭제

$( 선택자 ).parent()
바로 위에 존재하는 하나의 부모 요소 선택
JS 에서는 (요소).parentNode

$( 선택자 ).parents()
바로 위 부모뿐만 아니라 부모의 부모, 모든 조상을 선택

$( 선택자 ).children()
선택된 요소의 자식 요소 모두 선택

JS 에서는 (요소).childNodes 비 노드 요소 포함

JS 에서는 (요소).children 비 노드 요소 제외

$( 선택자 ).next()
선택된 요소 바로 다음에 위치한 형제 요소 선택
JS 에서는 (요소).nextSibling

$( 선택자 ).prev()
선택된 요소 바로 이전에 위치한 형제 요소 선택
JS 에서는 (요소).previousSibling

$( 선택자 ).addClass(“클래스명”)
선택된 요소에 클래스 추가

$( 선택자 ).removeClass(“클래스명”)
선택된 요소에 클래스 삭제

$( 선택자 ).hasClass(“클래스명”)
선택된 요소에 클래스가 있는지 여부 확인
Return 값은 true 와 false 로 나온다.
